# Git Cheat Sheet

| Command | Real-World Purpose | Terminal Command |
|---|---|---|
| Go up one folder | Move back to the parent directory | `cd ..` |
| List files | See what's inside the current folder | `ls` |
| Make a new folder | Create a new directory | `mkdir folder-name` |
| Clone a repo | Download a GitHub repo to your machine | `git clone <repo-url>` |
| Stage all changes | Mark all modified files ready to commit | `git add .` |
| Commit with a message | Save a snapshot of your changes with a description | `git commit -m "your message"` |
| Push to GitHub | Upload your committed changes to GitHub | `git push` |
